# Hermes Skills

This repository contains reusable Hermes Agent skills.

## Skills

- `skills/messaging/qq-bot-image-quiz` — QQ Bot quiz flows with question images plus InlineKeyboard buttons.

## Install

```bash
hermes skills install https://raw.githubusercontent.com/wyp1157/hermes-skills/main/skills/messaging/qq-bot-image-quiz/SKILL.md --name qq-bot-image-quiz
```
